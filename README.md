📊 Online Retail Sales Analysis using Python

📖 Overview

This project presents an end-to-end Exploratory Data Analysis (EDA) of an Online Retail dataset using Python. The objective was to transform raw transactional data into meaningful business insights through data cleaning, visualization, and statistical analysis.The analysis focuses on understanding sales performance, customer purchasing behavior, product trends, and revenue distribution to support business decision-making.

🎯 Business Objectives

- Identify top-performing products
- Analyze sales distribution
- Discover customer purchasing patterns
- Evaluate country-wise sales performance
- Identify revenue-driving products
- Understand pricing and quantity distribution

📂 Dataset

**Dataset:** Online Retail Dataset

The dataset contains transactional records from an online retail store, including:

- Invoice Number
- Product Description
- Quantity
- Unit Price
- Customer ID
- Invoice Date
- Country

**Source**
- UCI Machine Learning Repository
- Kaggle Online Retail Dataset

> The original dataset is not included due to GitHub file size limitations.

📌 Project Workflow

1️⃣ Data Cleaning

The raw dataset was cleaned by:

- Removing duplicate records
- Handling missing values
- Separating cancelled orders
- Removing invalid transactions
- Removing zero & negative prices
- Standardizing product descriptions
- Creating a Revenue column

2️⃣ Exploratory Data Analysis

Performed detailed analysis on:

- Dataset Overview
- Missing Values
- Duplicate Records
- Quantity Distribution
- Unit Price Distribution
- Revenue Distribution
- Country-wise Revenue
- Best Selling Products
- Least Selling Products
- Customer Purchase Behaviour
- Correlation Analysis
- Monthly Sales Trends

📈 Key Business Insights

✔ Identified countries generating the highest revenue.

✔ Identified top-selling and least-selling products.

✔ Analyzed customer purchasing behaviour.

✔ Evaluated pricing distribution across products.

✔ Identified sales trends using transaction history.

✔ Explored the relationship between Quantity, Unit Price and Revenue.
